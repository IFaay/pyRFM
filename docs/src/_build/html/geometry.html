Sample points inside the 3D polygon by: 1. Building a local orthonormal frame (e1, e2, n) for the plane. 2. Projecting all vertices to the (e1, e2) 2D coordinate system. 3. Using a Polygon2D to sample points in 2D. 4. Mapping the 2D samples back to 3D using the local frame.
