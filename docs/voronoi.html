interface_sample(num_samples: int) → Tuple[Dict, Tensor][source]

Samples points on the Voronoi interface (the boundaries between Voronoi cells). For each ridge (edge) in the Voronoi diagram:

  • If both vertices are valid (no -1), sample on the finite line segment.

  • If one vertex is -1, treat the edge as a ray and intersect with the domain boundary.

  • If both vertices are -1, treat it as an infinite line and intersect with the domain boundary.

Parameters:

num_samples (int) – Number of sample points to generate on each ridge segment.

Returns:

A collection of all sampled points on the Voronoi interfaces,

concatenated into a single tensor of shape (M, 2), where M depends on how many ridges intersect the domain.

Return type:

torch.Tensor
